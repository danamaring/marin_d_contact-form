# PHP CONTACT FORM

## Description

Develop a fully functioning PHP contact form.

## Getting Started

### Dependencies

* PHP
* Docker
* Mailhog

### Installing

* cd to the repo
* Compose-up Docker
* Open the contact form here: http://localhost:8080/php
* Open  Mailhog here: http://localhost:8025
